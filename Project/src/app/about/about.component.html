About

QuizBox is a simple full stack application (under development) which offers users access to trivia questions of their chosen category. The questions are provided by the Open Trivia Database (OpenTDB). Test your skills through 10 simple questions!

This program was developed by Adit Sinha. It uses front-end technologies like Angular and Bootstrap and back-end technologies like Node.js and Python. Read more about the development process and code below!
